# Activos Estáticos (Static Assets)

Utiliza la carpeta `/public` para almacenar todos los activos estáticos, como imágenes, fuentes, etc.

Todos los activos estáticos se sirven desde la raíz de tu servidor (`/` como en `https://ejemplo.com/`) y se superponen debajo de tus rutas (las rutas tienen prioridad).

```bash
public/
├── favicon.ico           # https://ejemplo.com/favicon.ico
└── images/
    └── logo.png          # https://ejemplo.com/images/logo.png
```
