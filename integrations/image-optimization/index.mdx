# Optimización de Imágenes

La optimización de imágenes es el proceso de reducir la cantidad de bytes que representan una imagen. Cuanto más pequeño sea el tamaño de la imagen, más rápido se cargará en una página. Cuanto más pequeño sea el tamaño de la imagen, menos ancho de banda consumirá. Menos ancho de banda consumido significa una mejor experiencia para tus usuarios, especialmente en redes móviles.


## `@unpic/qwik`

- Sitio con instrucciones detalladas de uso: [@unpic/qwik](https://unpic.pics/img/qwik/)
- Instalación: `npm install @unpic/qwik`

Unpic es una biblioteca de optimización de imágenes de terceros que funciona con CDN de optimización de imágenes existentes. Proporciona un componente `Image` que se puede utilizar para optimizar imágenes.

<CodeSandbox src="/src/routes/demo/integration/img/unpic/simple/index.tsx" style={{ height: '40em' }}>
```tsx {6} /Slot/
import { component$ } from '@builder.io/qwik';
import { Image } from '@unpic/qwik';

export default component$(() => {
  return (
    <Image
      src="https://cdn.shopify.com/static/sample-images/bath_grande_crop_center.jpeg"
      layout="constrained"
      width={800}
      height={600}
      alt="Un baño encantador"
    />
  );
});
```
</CodeSandbox>

> **Nota:** qwik-image y unpic no son un CDN y no alojan tus imágenes. Funcionan con CDN de optimización de imágenes existentes. Sugerimos usar algunos de los CDN populares:
> - Cloudinary
> - Cloudflare
> - Bunny.net
> - Vercel / Next.js
> - Imgix, incluyendo Unsplash, DatoCMS, Sanity y Prismic
> - Shopify
> - Kontent.ai
> - Builder.io
> - Contentful
> - Storyblok
> - WordPress.com y Jetpack Site Accelerator

## `qwik-image`

Imágenes eficientes con optimización automática.

```bash
npm install qwik-image
or
yarn install qwik-image
or
pnpm install qwik-image
```

> Este es un componente extensible para que los desarrolladores puedan conectar diferentes cargadores de imágenes a él (como builder.io u otros CDN).

<CodeSandbox src="/src/routes/demo/integration/img/qwik-image/index.tsx" style={{ height: '40em' }}>
```tsx {6} /Slot/
import { $, component$ } from '@builder.io/qwik';
import { Image, ImageTransformerProps, useImageProvider } from 'qwik-image';

export default component$(() => {
	const imageTransformer$ = $(
		({ src, width, height }: ImageTransformerProps): string => {
      // Aquí puedes establecer tu servicio de cargadores de imágenes favorito
			return `https://cdn.builder.io/api/v1/${src}?height=${height}&width=${width}}&format=webp&fit=fill`;
		}
	);

  // Proveedor Global (obligatorio)
	useImageProvider({
    // Puedes establecer esta propiedad para sobrescribir los valores predeterminados [3840, 1920, 1280, 960, 640]
    resolutions: [640],
		imageTransformer$,
	});

	return (
		<Image
			layout='constrained'
			objectFit='fill'
			width={400}
			height={500}
			alt='Paraíso tropical'
			placeholder='#e6e6e6'
			src={
				'image/assets%2FYJIGb4i01jvw0SRdL5Bt%2Fe5113e1c02db40e5bac75146fa46386f'
			}
		/>
	);
});
```
</CodeSandbox>
