# Storybook

[Storybook](https://storybook.js.org/) es un taller frontend para construir componentes de interfaz de usuario y páginas de forma aislada. Miles de equipos lo utilizan para el desarrollo, pruebas y documentación de la interfaz de usuario. Es de código abierto y gratuito.

Desde la versión 7.0, Storybook admite de forma nativa a Vite, lo que significa que también tiene soporte de primera clase para Qwik.

## Uso

Puedes agregar Storybook a tu aplicación o biblioteca existente usando el siguiente comando:

```shell
npm run qwik add storybook
```

El comando anterior instalará las dependencias necesarias y agregará un componente de ejemplo y sus historias.

Ahora puedes ejecutar el panel de control de Storybook:

```shell
npm run storybook
```
