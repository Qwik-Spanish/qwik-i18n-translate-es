# Íconos

Los íconos son una parte importante de cualquier aplicación. Ya hay más de 180,000 íconos que puedes agregar a tu aplicación Qwik.

## `qwikest/icons`

Este paquete permite agregar íconos de diferentes conjuntos de íconos a tu aplicación Qwik de una manera sencilla.

- `Bs`: Íconos de Bootstrap
- `Go`: Octicons de GitHub
- `Hi`: Heroicons de Tailwind
- `In`: Iconoir
- `Io`: Ionicons de Ionic
- `Lu`: Lucide [conjunto de íconos basado en feather icons]
- `Mo`: Mono Icons
- `Si`: Simple Icons [íconos de marcas populares]
- `Tb`: Tabler Icons

Simplemente instala el paquete con tu administrador de paquetes preferido:

```bash
npm i @qwikest/icons
yarn add @qwikest/icons
pnpm add @qwikest/icons
```

### Uso

```tsx
import { LuRocket } from "@qwikest/icons/lucide";

export const MyComponent = component$(() => {
  // El tamaño y color del ícono se heredan por defecto ⬇️
  return (
    <div style={{ color: "red", fontSize: "40px" }}>
      <LuRocket />
    </div>
  );
});
```

Por favor, consulta la [documentación oficial para más detalles](https://github.com/qwikest/icons).


## `icones.js.org`

[icones.js.org](https://icones.js.org/) es una colección de conjuntos de íconos. Hay más de 180,000 íconos de dominio público disponibles y listos para ser utilizados en tu aplicación Qwik.

Icones está impulsado por [iconify](https://iconify.design/), lo que permite agregar fácilmente íconos de Material Design, Phosphor, Remix, Carbon, Bootstrap, Tabler, Feather, Fluent, IconPark, Octicons y muchos otros conjuntos de íconos. También puedes agregar Twitter Emoji, Fluent Emoji, EmojiOne, Noto Emoji y otros a cualquier aplicación Qwik.
