# Tailwind

Tailwind es un framework CSS que nos proporciona clases de utilidad de un solo propósito que están estándarizadas en gran medida, y que nos ayudan a diseñar nuestras páginas web directamente desde dentro de nuestro marcado o desde archivos `.js/.jsx/.ts/.tsx/.mdx`. [Sitio web de Tailwindcss](https://tailwindcss.com/)

## Uso

Puedes agregar Tailwind fácilmente usando el siguiente script de inicialización de Qwik:

```shell
npm run qwik add tailwind
```

Este comando anterior actualiza tu aplicación con las dependencias necesarias.

También agrega nuevos archivos dentro de la carpeta de tu proyecto:

- `postcss.config.js`
- `tailwind.config.js`
- `.vscode/settings.json`

y modifica tu `src/global.css` para incluir

```css title="src/global.css"

# Archivo global.css

@tailwind base;
@tailwind components;
@tailwind utilities;

...código...

```
