# Nx y monorepos de escala empresarial

[Nx](https://nx.dev/) es una plataforma de desarrollo sólida y extensible diseñada para simplificar la gestión de monorepos de escala empresarial que contienen múltiples aplicaciones y bibliotecas, ofreciendo potentes herramientas para la generación de código, la orquestación de compilación, la gestión de dependencias y el intercambio de código.

Qwik se integra perfectamente con Nx utilizando el complemento [qwik-nx](https://github.com/qwikifiers/qwik-nx).


## Características principales

* Generar un nuevo espacio de trabajo Nx utilizando un preset de "Qwik".
* Generar aplicaciones y bibliotecas de Qwik.
* Generar componentes y rutas de Qwik.
* Generar Storybook, React Qwikify, configuración de Cloudflare y mucho más.
* Ejecutar ejecutores (executors) diseñados específicamente para compilar aplicaciones de Qwik.


## Uso

Puedes generar un nuevo espacio de trabajo ejecutando:

```shell
npx create-nx-workspace@latest org-workspace --preset=qwik-nx
```

O agregar una nueva aplicación de Qwik a un espacio de trabajo existente ejecutando:

```shell
npm install -D qwik-nx
```

y luego

```shell
nx generate qwik-nx:app
```

Para obtener más información, consulta la [documentación de qwik-nx](https://github.com/qwikifiers/qwik-nx).
