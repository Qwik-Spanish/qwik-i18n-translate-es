# Tauri

Tauri es un framework para construir aplicaciones de escritorio con cualquier framework de frontend y un núcleo (core) de Rust.

Obtén más información en el [sitio web de Tauri](https://tauri.app/v1/guides/getting-started/setup/qwik/).
