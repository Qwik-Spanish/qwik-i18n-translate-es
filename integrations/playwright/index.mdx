# Playwright

Playwright permite realizar pruebas end-to-end fiables para aplicaciones web modernas soportando todos los motores de renderizado modernos y funcionan en múltiples plataformas. [Sitio web de Playwright](https://playwright.dev/).

## Uso

Puedes agregar Playwright fácilmente utilizando el siguiente script de inicialización de Qwik:

```shell
npm run qwik add playwright
```

El comando anterior agrega Playwright a tus dependencias, añade un comando `test.e2e` en la sección de scripts de tu `package.json` y crea una configuración predeterminada dentro del archivo `playwright.config.ts` en el nivel raíz de la aplicación.
También agrega un nuevo archivo dentro de una nueva carpeta `tests` llamado `example.spec.ts`, que contiene algunos casos de prueba simples.

Para más información, consulta la [documentación de Playwright](https://playwright.dev/docs/intro).
