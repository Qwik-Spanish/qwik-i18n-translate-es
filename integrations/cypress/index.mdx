# Cypress

Con [Cypress](https://www.cypress.io), puedes crear fácilmente pruebas para tus aplicaciones web modernas, depurarlas visualmente y ejecutarlas automáticamente en tus compilaciones de integración continua.


## Pruebas de componentes

Cypress te permite probar tus componentes de Qwik de forma aislada con la ayuda del plugin de la comunidad [cypress-ct-qwik](https://github.com/qwikifiers/cypress-qwik).




## Uso

Puedes agregar Cypress fácilmente utilizando el siguiente script de inicio de Qwik:

```shell
npm run qwik add cypress
```

Este comando agrega `cypress` y `cypress-ct-qwik` a tus dependencias, añade comandos de prueba a la sección de scripts de tu `package.json` y crea los archivos de configuración necesarios para ejecutar pruebas de componentes con Cypress.


Para obtener más información, consulta la [documentación de Cypress](https://docs.cypress.io/guides/overview/why-cypress) y la [documentación de cypress-qwik](https://github.com/qwikifiers/cypress-qwik).
