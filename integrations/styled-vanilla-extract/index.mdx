# Styled Vanilla Extract

Styled-vanilla-extract es un framework de CSS construido sobre [vanilla-extract](https://vanilla-extract.style/) que proporciona un moderno y eficiente CSS-in-JS con Zero-runtime.

Esta integración admite dos formas de estilizar:

- Utilizando [Vanilla Extract](https://vanilla-extract.style/) directamente.
- [Styled-components](https://styled-components.com/) potenciado por Vanilla-extract.

## Uso

Puedes agregar `styled-vanilla-extract` fácilmente utilizando el siguiente script de inicialización de Qwik:

```shell
npm run qwik add styled-vanilla-extract
```

Después de la instalación, se creará una nueva ruta en tu proyecto que muestra el uso de la biblioteca en un nuevo componente.

### Forma Vanilla Extract

Por favor, consulta la documentación oficial para obtener más información sobre [Vanilla Extract](https://vanilla-extract.style/documentation/getting-started).

```tsx title="styles.css.ts"
import { style } from 'styled-vanilla-extract/qwik';

export const blueClass = style({
  display: 'block',
  width: '100%',
  height: '500px',
  background: 'blue',
});
```

```tsx title="component.tsx"
import { blueClass } from './styles.css';

export const Cmp = component$(() => {
  return <div class={blueClass} />;
});
```

```shell
npm run qwik add styled-vanilla-extract
```

> **¿Y qué hay de emotion u otras bibliotecas CSS-in-JS?** A pesar de su gran popularidad, emotion y otras bibliotecas CSS-in-JS no son la mejor elección para Qwik. [No están optimizadas para el rendimiento en tiempo de ejecución y no tienen un buen soporte de streaming SSR](https://dev.to/srmagura/why-were-breaking-up-wiht-css-in-js-4g9b), lo que conduce a un rendimiento de servidor y cliente degradado.

## Styled-components

Utilizando la misma integración, tienes styled-components potenciado por Vanilla Extract, utiliza la función exportada `styled` para crear tus componentes.

```tsx title="styles.css.ts"
import { styled } from 'styled-vanilla-extract/qwik';

export const BlueBox = styled.div`
  display: block;
  width: 100%;
  height: 500px;
  background: blue;
`;
```

```tsx title="component.tsx"
import { BlueBox } from './styles.css';

export const Cmp = component$(() => {
  return <BlueBox />;
});
```
