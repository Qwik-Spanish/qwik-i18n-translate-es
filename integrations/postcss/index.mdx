# PostCSS

[PostCSS](https://postcss.org/) permite a los desarrolladores escribir CSS moderno manteniendo la compatibilidad con versiones anteriores y mejorando el rendimiento. Analiza el código CSS y lo pasa a través de una serie de complementos que pueden modificar, optimizar y extender el CSS antes de generar el resultado final.

## Uso

Puedes agregar PostCSS a tu proyecto utilizando el siguiente script de inicialización de Qwik:

```shell
npm run qwik add postcss
```

Esto creará un nuevo archivo `postcss.config.js` en la raíz del proyecto con las siguientes dependencias y configuraciones.

- [Autoprefixer](https://github.com/postcss/autoprefixer)
- [PostCSS Preset Env](https://preset-env.cssdb.org/)

```ts title="postcss.config.js"
module.exports = {
  plugins: {
    autoprefixer: {},
    "postcss-preset-env": {
      stage: 3,
      features: {
        "nesting-rules": true,
      },
    },
  },
}
```

Si deseas agregar un nuevo complemento como [CssNano](https://cssnano.co/), ejecuta el siguiente comando y actualiza el archivo `postcss.config.js`.

```shell
npm install cssnano --save-dev
```

```ts title="postcss.config.js"
module.exports = {
  plugins: {
    /* configuración del complemento anterior */
    "cssnano": {
      preset: "default"
    }
  },
}
```

Para más documentación, consulta [PostCSS Docs](https://postcss.org/docs/)
