# Te damos la bienvenida a Qwik

Qwik es un nuevo tipo de framework web que puede ofrecer aplicaciones web de carga instantánea de cualquier tamaño o complejidad. Sus sitios y aplicaciones pueden arrancar con aproximadamente 1kb de JS (independientemente de la complejidad de la aplicación) y lograr un rendimiento constante a escala.

## Metas de Qwik

<div class="card-grid">
  <div class="card">
    <h3>De uso general</h3>
    <p>
      Qwik puede utilizarse para crear cualquier tipo de sitio web o aplicación.
    </p>
  </div>
  <div class="card">
    <h3>Activación instantánea</h3>
    <p>
      A diferencia de otros frameworks, Qwik es reanudable lo que significa que
      las aplicaciones Qwik requieren 0 hidratación. Esto permite que las
      aplicaciones Qwik tengan interactividad instantánea, independientemente de
      su tamaño o complejidad.
    </p>
  </div>
  <div class="card">
    <h3>Optimizado para la velocidad</h3>
    <p>
      Qwik tiene un rendimiento sin precedentes: ofrece cargas de página
      completa por debajo del segundo, incluso en dispositivos móviles. Lo logra
      mediante al entregar tan solo puro HTML, y realizando la carga incremental
      de JS únicamente según sea necesario.
    </p>
  </div>
</div>
